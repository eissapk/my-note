# My Note
My Note is a flexible and free web app to organize your plans, projects, and more.

## Features
* Progressive Web App
* Synced With LocalStorage
* Upload and Backup Your Data
* Add Task
* Remove - Edit Task
* Tasks Counter
* Dark Mode
* Animations
* Fully Responsive
* No Frameworks - No Libraries
* Fast Loading
* Cross-Browser
* Much More

## Built With
* HTML5
* CSS3
* Pugjs
* Sass
* RWD - Grid System
* Vanilla JS - OOP
* JSON
* Gulp
